# Flipword trailer made with Remotion

## Commands

**Install Dependencies**

```console
yarn
```

**Start Preview**

```console
yarn start
```

**Render MP4 video**

```console
yarn build
```

**Upgrade Remotion**

```console
yarn run upgrade
```
